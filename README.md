# 🏪 US Retail Performance Analytics (2017–2021)

📌 A relational database project engineered to analyze US retail sales performance by mapping order metrics across store locations, regional territories, and sales teams.

---

# 📊 Project Overview

This project analyzes a US Regional Sales dataset from the Retail and Wholesale Distribution industry. The objective was to transform raw operational and transactional data into strategic business insights that drive profitability, optimize supply chain logistics, and maximize regional sales efficiency.

The analysis supports executive leadership and marketing teams by uncovering sales patterns, evaluating regional performance, and identifying opportunities for targeted retail expansion.

# 🛠 Data Preparation

The following preprocessing steps were performed:

1. Converted Excel workbook files into CSV format for MySQL compatibility.
2. Removed commas and whitespace from Unit Price and Unit Cost columns.
3. Converted text-based numerical fields into decimal data types.
4. Validated data structure before analysis.
```sql
-- Removing the whitespaces and commas from the unit price and unit cost so that the datatype can be changed
Update `sales order_usa`
Set 
`Unit Price` = trim(replace(replace(`Unit Price`,'$',''),',','')),
`Unit Cost` = trim(replace(replace(`Unit Cost`,'$',''),',',''));

-- Changing the datatype from text to decimal
Alter table `sales order_usa`
Modify column `Unit Price` decimal(19,2),
Modify column `Unit Cost` decimal(19,2);
```

# 📊 Analysis Performed

## 1. Regional Profit Analysis

Determined total profit generated by each region and identified the highest-performing region.

### SQL Query

```sql
Select `sales team_usa`.Region,round(sum(`sales order_usa`.`Unit Price`*`Order Quantity`*(1-`Discount Applied`)-(`Unit Cost`*`Order Quantity`))) as Total_Profit
from `sales team_usa` 
left join `sales order_usa` on
`sales team_usa`._SalesTeamID =`sales order_usa`._SalesTeamID
group by Region
order by Region asc;
```

## 2. Product Contribution to Regional Profit

Identified the top-performing product within each region using CTEs and ranking functions.

### SQL Query

```sql
With RankedData as(
Select `sales order_usa`._ProductID,`sales team_usa`.Region,
rank() over (partition by `sales team_usa`.Region order by sum(`sales order_usa`.`Unit Price`*`Order Quantity`*(1-`Discount Applied`)-(`Unit Cost`*`Order Quantity`))desc) as Profit_rank
from `sales team_usa` 
right join `sales order_usa` on
`sales team_usa`._SalesTeamID =`sales order_usa`._SalesTeamID
group by Region, _ProductID
)
select _ProductID,Region, Profit_rank from RankedData
group by Region, _ProductID
having Profit_rank = 1;
```

`sales team_usa`._SalesTeamID =`sales order_usa`._SalesTeamID
